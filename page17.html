Entregable N°6

Matrices de evaluación y modelado 3D

Recapitulación de las semanas anteriores

Problema: Falta de tecnología personalizada y accesible para el monitoreo de las señales pulmonares y cardiacas de pacientes asintomáticos a distancia
Estrategia: Banda para la cabeza capaz de medir temperatura y pulsioximetría mediante señales ppg.
Concepto: M.A.P.A. (Mediciones Ayudando al Paciente Asintomático)
Pila de reloj -> Botón táctil -> Sensor DS18B20+ -> Sensor PPG MAX30102 -> Arduino Mini Pro -> Bluetooth HC-05 -> APP desarrollada en Java
Proyectos preliminares:
1) Estilo antifaz, orientada para el uso meintras duerme el paciente
2) Estilo deportivo, orientada al uso diario
3) Estilo casual, ideal para el uso diario.

Tarjeta

Para los componentes que vamos a utilizar desarrollamos una tarjeta en kiCAD. Esta incluye todos los componentes listados en el concepto ganador, así como las pilas. 
Dimensiones (sin considerar tamaño de los sensores):
-Largo: 78.169 milímetros
-Ancho: 38.925 milímetros

También fue subida al sitio p3d.in para su adecuada visualización. Para ver el modelo, se lo puede rotar con el clic izquierdo y mover con el clic derecho.
Como se ve, no algunos sensores no se encuentran en el modelo. Esto se debe a disponibilidad, ya que ni el programa ni repositiorios externos (grabcad, 3Dcontentcentral) cuentan con estos disponibles para usar.
